# Strepthanthus tortuosus Genomic offset simulations
Using existing allele frequency of the various Streptanthus tortuosus populations and their genomic
offset calculations, we simulate admixture scenarios between genetically diverse donor
populations and genetically constrained recipient populations. Specifically, we perform:
 1) identify compatible donor-recipient pairs based on genetic diversity, genomic offset, and population genetic distance (FST)
 2) simulate allele frequency mixing at varying admixture levels (5-30%) and recalculate genomic offset for admixed populations
 3) evaluate rescue effectiveness across climate scenarios
 4) assess risks including outbreeding depression and selection against immigrant alleles.
